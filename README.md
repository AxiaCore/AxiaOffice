## Keeping wifi up

Put the script ```keepalivewifi/keepalivewifi.sh``` at ```/usr/local/bin/``` then:

$ sudo chmod +x /usr/local/bin/keepalive.sh

Next edit the crontab file and add the content from ```keepalivewifi/crontab``` into it
